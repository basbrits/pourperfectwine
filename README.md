# RijksLens Prototype Debug Version

This is a static web prototype for recognizing one artwork and opening a tappable artwork view.

Edit content in:

`data/artworks.js`

The three categories are:

- history
- technique
- symbolism

Each bubble has `x` and `y` coordinates in percentages.

To find coordinates, open the site with `?edit=1`, open the demo artwork, and tap the painting.
